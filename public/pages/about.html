Minigame Mayhem

O Minigame Mayhem é um site onde você pode jogar diversos jogos.
É um site ideal para se para passar o tempo jogando desde clássicos a jogos mais modernos.

O objetivo desse site é a prática das linguagens HTML, CSS e JavaScript, além do Express, NPM e NodeJS.
